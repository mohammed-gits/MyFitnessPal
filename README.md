# MyFitnessPal Application

### CSCI 410 Project â€“ Mobile Development
A modern and simple workout-tracking app built with Flutter.

This application helps users manage their workout routines, track body stats, and use a built-in rest timer for efficient gym sessions.  
In **Phase 2**, the app is fully integrated with a **PHP + MySQL backend** and deployed using **AwardSpace hosting**.

---

## Features
- Custom workout splits (stored in MySQL)
- Add, edit, and delete exercises per workout
- Track sets completion per exercise
- Workout progress calculation
- Rest timer with animations
- Body stats tracking (weight, waist, etc.)
- Backend API integration (PHP + MySQL)
- Clean dark theme design
